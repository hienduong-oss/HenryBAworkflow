# Collaboration

Section này dành cho cách vận hành BA-kit khi có nhiều người cùng làm trên một initiative.

Trọng tâm của section:

- BA Lead làm gì trước
- BA Member được phép làm gì theo từng module
- Quy ước branch và pull request
- Cách review, merge và package
- Cách phân quyền GitHub để quy trình này chạy ổn định
